# BOW_imdb_dataset

This project demonstrates the application of the **Bag of Words (BoW)** technique on the
**IMDB Movie Reviews Dataset** for text preprocessing and feature extraction in
Natural Language Processing (NLP) tasks.

---

##  Project Overview
The goal of this project is to convert raw movie review text into numerical features
using the Bag of Words model.  
These features can later be used for machine learning tasks such as sentiment
classification.

---

##  Dataset
- **IMDB Movie Reviews Dataset**
- 50,000 movie reviews
- Binary sentiment labels:
  - Positive
  - Negative

---

##  Techniques Used
- Tokenization
- Bag of Words (Count Vectorization)

---

## Tools & Libraries
- Python
- NumPy
- Pandas
- Scikit-learn
